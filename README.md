# XRP Trading Dashboard (MVP)

### Project Overview
Professional trading simulation and monitoring tool for XRP/USD. Developed in Google AI Studio to provide data-driven decision support and automated risk calculation.

### Core Functionality
* **Market Monitoring:** Real-time XRP price tracking and price action alerts.
* **Strategy Module:** Backtesting engine supporting hourly (1H) and daily data sets.
* **Risk Engine:** Automated position sizing based on equity risk (default 2% rule).
* **UI/UX:** High-contrast dark mode optimized for macOS and mobile (iPhone/iPad).

### Technical Specifications
* **Frontend:** React / Tailwind CSS / Lucide Icons.
* **Logic:** Python & Node.js integration for PnL and technical indicators.
* **Execution:** Rules-based checklists for trade entry and exit.

### Decision Support Framework
- **Data Integrity:** Strict reliance on verified price data; no synthetic/invented values.
- **Position Sizing:** Mandatory stop-loss and capital allocation logic.
- **Verification:** Clear distinction between historical knowns and strategy assumptions.
